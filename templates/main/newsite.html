Tips
  • The wsgi application entry point should be a variable called "app" in a file called "app.py"
  • You must use an absolute path to refer to your sqllite database. Use os.path.join(os.path.dirname(__file__), "mydatabase.db") to calculate the absolute path.
  • Make sure that the application works on your computer and that the database has been initialized.
  • Do not hard-code ANY urls. Do reverse url lookups for all links, images, stylesheets, etc.
  • Max filesize is around 10 mb
  • This website intentionally exposes debug information of all websites. Do not use this information to interfere with other people's websites.
